IDE ou Editor de Texto?

Para desenvolver qualquer software, basta um editor de texto e um compilador

Editor de Texto

  • Interface de simples utilização
  • Consome pouco recurso da máquina
  • Fácil configuração

IDE

  • Integrated Development Environment
  • Integrada!!
  • Ferramentas de assistência ao desenvolvimento
  • Automatização de procedimentos

IDE no Ecossistema Android

Desenvolver apps móveis requer uma grande quantidade de ferramentas!

Android SDK: framework, bibliotecas, debugger, emulador, etc

Portanto o uso de uma IDE é fundamental!

IDE Android

  • Eclipse ADT: plugin independente disponibilizado pela Google para ser integrado ao Eclipse
    • Depreciado em 2015!
  • Android Studio: IDE oficial desde 2015, resultado de parceria entre Google e Jet Brains

Android Studio

IDE oficial para desenvolvimento Android, baseado no IntelliJ IDEA

  1. Sistema de build automatizado
  2. Emulador integrado
  3. Ambiente único para desenvolvimento em vários dispositivos (Telefone, carro, wearables, etc)
  4. Hot-reloading (realmente muito útil!!)
  5. Integração com sistemas de controle de versão, inclusive GitHub!
  6. Suite de ferramentas para testes e debugging

Estilo e Formatação

Pode ser customizável em File > Settings > Editor > Code Style

Descrição Tecla de Atalho
Formatar trecho específico ctrl + alt + L

Debugging

Uma das grandes vantagens em se utilizar o Android Studio: suporte a várias ferramentas de debugging e profiling de programas:

  1. Inline debugging
  2. Profiler de performance
  3. Profiler de memória
  4. Análise estática de código (Linters)

Logcat

Painel para exibição e análise de mensagens de log impressos durante execução da aplicação.

  1. Limpar
  2. Scroll de mensagem
  3. Quebra de linhas
  4. Imprimir
  5. Reiniciar
  6. Customizar
  7. Captura de tela
  8. Gravação de vídeo
